In this final year project at HKUST, we developed a robot assistant that uses Vision-Language-Action (VLA) models to control a robot arm.

Throughout the entire project, we:

- Developed a web application powered by a LLM to interface with the robot arm.
- Developed a Gazebo simulator to simulate and verify the robot's trajectory.
- Collected the training data and fine-tuned the VLA model.
- Contribute the collected datasets to [open-source community](https://huggingface.co/Thomas-Yip/datasets) (> 1k downloads)

[Demo video](https://www.youtube.com/watch?v=QZBQsq89Mpk)

**Skills Earned:** Vision-Language-Action (VLA) Models, Machine Learning Fine-Tuning, Agentic AI
